# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]
### Added
- APIs related to 
  - write transactions (counter ceilings, key parameters)
  - asynchronous transactions in static mode (export / import write commands)
### Upgraded
- "Calypsonet Terminal Reader API" to version `1.2.0`.

## [0.1.0] - 2022-12-12
### Added
- APIs of the SAM image and SAM selection.
- APIs related to signature computation and verification as well as counter and ceiling reading.
